# Stranger Things

A React + Vite web application featuring an immersive Stranger Things themed experience. This project showcases animated UI components, smooth transitions, and a dark atmospheric design.

## Features

- 🎬 **Animated Hero Section** - Dynamic animations powered by Framer Motion
- 🌙 **Dark Theme** - Mysterious Stranger Things aesthetic
- 📱 **Responsive Design** - Works seamlessly on all devices
- ⚡ **Vite** - Lightning-fast development and build process
- ✨ **Smooth Animations** - Mouse-tracking effects and page transitions
- 📋 **Navigation** - Easy access to Home, Characters, Episodes, and Upside Down

## Project Structure

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

### Build

```bash
npm run build
```

### Lint

```bash
npm run lint
```

## Technologies Used

- **React** - UI framework
- **Vite** - Build tool and dev server
- **Framer Motion** - Animation library
- **CSS3** - Styling and effects
